2. `research_pack`
3. result examples

目标：
1. 补足 `HUGE / MAX` 的完整 preset、result 和运行时拓扑

仍需实现：

2. 项目级分层协作

1. `LocalHttpBridge`
2. bridge launcher / serve 入口
3. bootstrap / capability sync / reconnect 的明确责任切分
